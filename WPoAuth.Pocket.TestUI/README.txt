Pocket - https://getpocket.com/developer/
=================================================================
A library for performing oAuth 2.0 requests - can be integrated into Windows Phone apps.
Based on code at http://developer.nokia.com/community/wiki/OAuth_on_Windows_Phone.
=================================================================

Request Endpoint = https://getpocket.com/v3/oauth/request
Authorization Endpoint = https://getpocket.com/v3/oauth/authorize
Authorization Web Url = https://getpocket.com/auth/authorize

* Get a consumer key from the Url at the top of the page and paste it into the Test UI.
